## Laboratorium 1: Git, GitHub i przygotowanie środowiska
**Opanowanie systemu kontroli wersji Git, platformy GitHub oraz przygotowanie lokalnego środowiska programistycznego dla wybranego frameworka (np. Django, React). Szczególny nacisk położono na poprawne dokumentowanie pracy przy użyciu Markdown oraz konfigurację bezpiecznego dostępu przez SSH.**

* Konfiguracja środowiska i Markdown
* Inicjalizacja projektu i Git
* Praca z gałęziami i podstawowa logika
* Współpraca z GitHub
* Symulacja konfliktu 
* Automatyzacja z GitHub Actions
